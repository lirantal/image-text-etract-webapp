💡 Tips for Better Detection

  • Good Lighting: Ensure the text is well-lit
  • Stable Camera: Hold the camera steady
  • Proper Distance: Not too close, not too far
  • Clear Background: Avoid cluttered backgrounds
  • Text Orientation: Keep text horizontal
  • Debug Mode: Enable debug mode to see processing details

⚙️ Performance Settings

In the main application, try adjusting these settings:

  • Detection Interval: Lower values (100-200ms) for faster response
  • Confidence Threshold: Lower values (0.3-0.5) for more sensitive detection
  • Debug Mode: Enable to see detailed processing information
